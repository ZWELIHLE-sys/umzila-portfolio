# Images for the website

Save your photos in THIS folder, using these EXACT file names (all lowercase):

| File name        | What it is                                  | Used where            |
| ---------------- | ------------------------------------------- | --------------------- |
| `hero-cave.jpg`  | The cave / light-rays travel photo          | Homepage hero background |
| `zwelihle.jpg`   | YOUR personal photo (portrait works best)   | Homepage hero (glass frame) |
| `vr-tech.jpg`    | The person holding the VR headset           | Homepage "Vision" section |

Tips:
- `.jpg`, `.jpeg`, `.png` and `.webp` all work — but keep the name (e.g. `hero-cave`)
  exactly as above. If you use a `.png`, update the extension in the code too (just ask me).
- Bigger is fine — Next.js automatically optimises and resizes them.
- After dropping the files here, refresh http://localhost:3000 to see them appear.
